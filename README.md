## Project Overview
This repository contains a full set of business data analysis reports derived from a single master dataset. The project utilizes **Pivot Tables** in Excel (saved here as CSVs) to transform raw transactional data into key performance indicators (KPIs) and insightful trends across various business dimensions.

## üìÇ Repository Structure

| File Type | File Name Example | Purpose |
| :--- | :--- | :--- |
| **Master Data** | `EXCEL PROJECT.xlsx - MASTER DATA.csv` | The raw, clean, transactional dataset used for all analysis. |
| **Analysis Reports** | `EXCEL PROJECT.xlsx - PIVOT TABLE -[#].csv` | Nine distinct reports summarizing key business metrics. |

## üéØ Key Analysis Reports (Pivot Tables)
The pivot tables below provide focused summaries on different aspects of the business data.

| Report File | Likely Analysis Focus |
| :--- | :--- |
| `PIVOT TABLE -1.csv` | Overall Sales/Revenue Summary or Top-Level Totals |
| `PIVOT TABLE -2.csv` | Breakdown by Primary Category (e.g., Product Type or Region) |
| `PIVOT TABLE-3.csv` | Trend Analysis over Time (e.g., Monthly/Quarterly Performance) |
| `PIVOT TABLE -4.csv` | Performance by Secondary Category (e.g., Sales Rep or Customer Segment) |
| `PIVOT TABLE- 5.csv` | Comparative Analysis (e.g., Actual vs. Budget, or Year-over-Year) |
| `PIVOT TABLE-6.csv` | Detailed Profitability Metrics or Margin Analysis |
| `PIVOT TABLE- 7 .csv` | Geographical Breakdown (e.g., City or State Performance) |
| `PIVOT TABLE- 8.csv` | Customer/Client Segmentation or Count of Transactions |
| `PIVOT TABLE- 9.csv` | Custom Drill-down or Specific Filtered View |
